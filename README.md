# Terraform na Prática

### 01 - Instale o terraform 
[click here and install](https://www.terraform.io/downloads.html)



### 02 - Criando site estatico

<img src="https://test-highlander-s3.s3-sa-east-1.amazonaws.com/site-estatico.png" width="60%" valign="top" />

##### Inicializando os recursos 

```
terraform init
terraform plan
terraform apply -auto-approve
```

##### Destruindo os  recursos 

```
terraform destroy -auto-approve
```
